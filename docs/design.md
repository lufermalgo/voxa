# Design System Documentation

This document outlines the core design principles and token values that define our digital products' aesthetic and functional aspects.

## Brand Identity

### Color Palette
Our brand colors are integral to our identity, ensuring a consistent and recognizable user experience.
- **Primary Color:** `#9D85FF` - The most distinctive chromatic color, used for key interactive elements and calls to action.
- **Secondary Color:** `#5078e8` - A supporting color for less prominent UI elements and secondary actions.
- **Tertiary Color:** `#C0C0C0` - An additional accent color for highlights and decorative elements.
- **Neutral Color:** `#fbfbff` - A neutral base color for backgrounds, surfaces, and non-chromatic elements, providing balance and contrast.

Our interface operates in a **light** color mode, providing a bright and airy feel across all applications.

### Typography
We use a distinct set of fonts to establish a clear typographic hierarchy and enhance readability.
- **Headline Font:** `Manrope` - Used for all major headings and titles, conveying a modern and sophisticated tone.
- **Body Font:** `Inter` - Chosen for its excellent readability across various screen sizes, ideal for all body text.
- **Label Font:** `Manrope` - Applied to labels, buttons, and other interactive text elements for consistency with our headlines.

## UI Elements

### Roundedness
UI elements feature **maximum, pill-shaped** roundedness, contributing to a soft and friendly visual language.

### Spacing
We maintain a **normal** level of whitespace, providing a balanced and comfortable density for content and interactive elements.
