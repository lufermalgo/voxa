# Voxa Brand Identity

Official brand specifications and design guidelines for Voxa.

## 1. Visual Essence
Voxa is minimalist, efficient, and "airy". It represents the fluid transition from voice to text.

![Voxa Proportional Logo](file:///Users/lufermalgo/.gemini/antigravity/brain/20d02415-72f6-4f98-b5bd-80352457c65e/voxa_app_icon_proportional_1775483093401.png)

## 2. Color Palette
- **Primary (Voxa Violet)**: `#9D7AFF` (approximate based on user preference)
    - *Note*: Use for the app icon background and primary UI call-to-actions.
- **Secondary (Pure White)**: `#FFFFFF`
    - *Note*: Used for the logo centerpiece (sound bars).

## 3. Typography
- **Primary Font**: `Inter` or `Outfit` (Modern, clean, sans-serif).
- **Tone**: Professional yet approachable.

## 4. Logo Specifications

### A. Main App Icon (Dock)
- **Shape**: macOS Squircle (Rounded square with continuous curvature).
- **Background**: Solid Voxa Violet.
- **Centerpiece**: 5 vertical white bars of varying heights (Sound Wave pattern).
- **Proportions**: The sound wave should occupy approximately 60-70% of the icon's visual height to ensure clarity at small sizes.

### B. Menubar Icon (Tray)
- **Style**: Template image (Monochrome).
- **Design**: Simplified sound wave (3-5 bars) without background.
- **Size**: 16x16px (visual), 22x22px (canvas).

### C. Landing Page Logo
- **Layout**: Horizontal.
- **Elements**: [Wordmark "Voxa"] + [Minimal Sound Wave Icon].
- **Spacing**: Generous letter-spacing for the wordmark.
